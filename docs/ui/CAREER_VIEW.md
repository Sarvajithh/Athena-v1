# Career View

### Purpose
The **long-horizon thread** — internships, applications, skills, portfolio — kept visible but never urgent. This screen should feel like looking at a slow-growing plant, not a sales pipeline.

---

## Layout

### Goal Rings (top)
- Up to 3 active long-term goals (e.g., "Summer Internship," "Portfolio Project," "Certification"), each shown as a large soft progress ring with a single icon in the center — no percentage number by default (tap to reveal exact %, Calm mode hides it).
- Only 3 max shown at once — anything else lives in a quiet "Someday" list below, deliberately out of the main sightline to avoid overload from too many open-ended goals.

### Stage Board (middle)
- A horizontal Kanban-style board, **Linear-flavored**: columns are soft, minimal, icon-labeled (e.g., Researching → Applying → Interviewing → Offer), cards are compact with just a company/role icon and name.
- Cards drag between columns with the same snap-motion as Weekly Planner blocks — visual consistency across the app reinforces "this is all one calm system."
- No column shows a count badge unless it's non-zero, and never in red.

### Skills / Growth strip (bottom)
- A soft horizontal row of skill tags the user is building this semester, each as a small filled/unfilled progress dot-set (like Apple Watch activity dots, but muted colors) — purely visual growth tracking, no percentages, no comparisons to others.

---

## Interaction philosophy
- This screen is **read-mostly**. It updates as a reflection of what's happening elsewhere (finishing a portfolio task in Daily Planner nudges the relevant ring), rather than demanding the user manage it directly every day.
- No deadlines shown here with urgency styling — Career View is intentionally the calmest screen in the app, since long-term goals should never feel like today's pressure.

## Empty state
- No active goals yet → a single calm illustration + one soft button: "Set a direction." One tap opens a minimal 3-field flow (icon, name, target date) — nothing more.

---

## Inspiration notes
- **Apple Watch/Fitness**: dot-based growth tracking, ring metaphor for long-term progress.
- **Linear**: kanban board minimalism, drag-and-snap interaction.
- **Notion**: flexible "Someday" list as an overflow container, keeping the main view sparse.
- **Arc**: consistent icon language carried through from other views.
