# tests/e2e/

Reserves the location for Playwright end-to-end tests
(`IMPLEMENTATION_PLAN.md` §11.1). Sprint S01 (Foundation Scaffold) has no
real user flow to drive — no product screens exist yet
(`SPRINT1_SPEC.md` §0) — so no Playwright test or dependency is added
yet.

The first end-to-end suite is added by the first sprint that ships a
real user flow (e.g. Semester Setup → logging → Now screen ranking),
per `SPRINT1_SPEC.md` §7 ("Explicitly not yet automated: End-to-end
(Playwright) — no user flow exists yet to drive").
